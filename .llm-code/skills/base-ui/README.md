# Base UI Skill

Agent skill for building React applications with Base UI (@base-ui/react) unstyled components.

## Usage

Use this skill when:

- Building UI with `@base-ui/react` primitives
- Composing unstyled components with custom markup and styling
- Implementing menus, overlays, toasts, and toolbars
- Building accessible form controls
