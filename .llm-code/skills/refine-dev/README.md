# Refine.dev Skill

Build React admin applications with [Refine](https://refine.dev) - a headless CRUD framework.

## Scope

- **Build**: Vite (not Next.js/CRA)
- **UI**: Mantine v5
- **Router**: React Router

## What This Skill Covers

- Data providers for REST/GraphQL backends
- CRUD hooks (useList, useForm, useTable, useSelect)
- Authentication & access control (RBAC/ABAC)
- Mantine UI integration
- i18n, notifications, realtime
