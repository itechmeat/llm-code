# FastAPI Skill

A skill for building modern, fast web APIs with FastAPI framework.

## What It Does

- Guides creation of REST APIs with Python and FastAPI
- Covers request/response handling with Pydantic models
- Explains authentication (OAuth2, JWT)
- Provides patterns for dependencies and middleware
- Covers testing and project structure
