# Inworld AI Skill

Agent skill for text-to-speech (TTS) with Inworld AI platform.

## Usage

Use this skill when:

- Generating speech from text with Inworld TTS API
- Cloning voices from audio samples
- Adding emotion and delivery style to speech
- Implementing lip sync with timestamps/visemes
- Using custom pronunciation (IPA)
