# Mantine UI Skill

A comprehensive guide for building React applications with [Mantine](https://mantine.dev) — a fully-featured UI library with 100+ components and hooks.

## What This Skill Covers

- **Setup**: Vite + TypeScript configuration, PostCSS preset
- **Components**: Layouts, inputs, overlays, navigation, data display
- **Styling**: CSS modules, style props, theming, dark mode
- **Forms**: useForm hook with validation (Zod/Yup support)
- **Hooks**: 75+ utility hooks for common patterns
- **Testing**: Vitest setup with custom render
- **Linting**: eslint-config-mantine
