# Qdrant Skill

Operational playbook for Qdrant vector database: collections, points, payload filtering, indexing, deployment, and performance tuning.

## What This Skill Covers

- **Core Concepts**: Collections, points, payload, vectors, filtering
- **Data Operations**: CRUD, batch ingestion, upsert semantics
- **Retrieval**: Search, filtering, hybrid queries, explore/recommendations
- **Performance**: Indexing strategies, HNSW tuning, quantization, optimizer
- **Deployment**: Docker, Kubernetes, storage configuration, security/TLS
- **Operations**: Snapshots/backup, monitoring, troubleshooting

Activate this skill when working with Qdrant vector storage: creating collections, ingesting embeddings, tuning search performance, configuring deployment, or troubleshooting issues.
