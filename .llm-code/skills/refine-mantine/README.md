# Refine + Mantine Integration

Build admin panels with [Refine](https://refine.dev) + [Mantine](https://v5.mantine.dev/) UI.

## Scope

- `@refinedev/mantine` components (layout, views, forms, buttons, fields)
- `@refinedev/react-table` for data tables
- **Mantine v5 only** (required by Refine integration)

## What This Skill Covers

- App setup with ThemedLayoutV2
- CRUD views (List, Show, Edit, Create)
- Form hooks (useForm, useModalForm, useDrawerForm)
- Table integration with TanStack Table
- Auth pages and theming
