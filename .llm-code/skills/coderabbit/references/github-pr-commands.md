# GitHub PR commands (@coderabbitai)

Use `@coderabbitai` in pull request comments (supported features vary by plan).

| Command | Description |
| --- | --- |
| `@coderabbitai review` | Incremental review of new changes |
| `@coderabbitai full review` | Complete review from scratch |
| `@coderabbitai pause` | Stop automatic reviews |
| `@coderabbitai resume` | Restart reviews |
| `@coderabbitai summary` | Regenerate PR summary |
| `@coderabbitai resolve` | Mark conversations as resolved |
| `@coderabbitai configuration` | Show current settings |

Note: Treat PR comment commands as optional helpers. Local CLI review is the primary workflow covered by these skills.
