# OpenAPI Skill

A skill for creating, validating, and working with OpenAPI Specification (OAS) documents.

## What It Does

- Guides creation of OpenAPI 3.x specification documents
- Provides recipes for common API patterns
- Explains schemas, parameters, security, and components
- Helps validate and troubleshoot OpenAPI documents
