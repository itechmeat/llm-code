# Installation

This skill assumes Zvec is already installed. Use this page only when you need to set up Zvec in a new environment.

## Python

- Requirement (as documented in the GitHub README): Python 3.10–3.14

```bash
pip install zvec
```

## Node.js

```bash
npm install @zvec/zvec
```
