# Vibe Kanban Skill

Orchestration platform for AI coding agents with isolated git worktrees, multi-agent support, and visual code review.

## What This Skill Covers

- **Installation & Setup**: Prerequisites, running via npx, initial configuration
- **Workspaces (Beta)**: Sessions, command bar, multi-repo, workspace notes, integrated terminal
- **Core Features**: Projects, tasks, monitoring execution, testing, code review, completing tasks
- **Advanced Features**: Subtasks, new attempts, resolving rebase conflicts
- **Configuration**: Global settings, agent profiles/variants, task tags, keyboard shortcuts
- **Integrations**: GitHub, Azure Repos, VSCode Extension, MCP servers
- **Best Practices**: Vibe Guide tips for effective AI agent workflows
- **Troubleshooting**: Common issues and solutions

## When to Use

Activate this skill when you need to:

- Set up Vibe Kanban for AI-assisted development
- Manage multiple coding agents (Claude Code, Codex, Gemini, Antigravity, etc.)
- Create and execute tasks in isolated git worktrees
- Work with Workspaces: sessions, multi-repo, command bar
- Review AI-generated code changes
- Configure agent profiles and MCP integrations
