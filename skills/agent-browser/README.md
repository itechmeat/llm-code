# Agent Browser Skill

Headless browser automation CLI for AI agents.

## What This Skill Covers

- Installation and setup
- 50+ commands for navigation, forms, screenshots, network, storage
- Refs (element references) for deterministic interaction
- Sessions for isolated browser instances
- Snapshots (accessibility tree) for AI-friendly output
- Streaming for live viewport preview
- Agent mode integration patterns
- CDP mode for connecting to existing browsers

## Quick Start

```bash
npm install -g agent-browser

agent-browser open example.com
agent-browser snapshot -i
agent-browser click @e2
agent-browser close
```
