# Telegram Bot Skill

A comprehensive guide for building Telegram bots with [aiogram 3](https://docs.aiogram.dev/) — an async Python framework.

## What This Skill Covers

- **Bot API**: Commands, messages, inline mode, keyboards
- **Webhooks**: Setup, security, idempotency patterns
- **aiogram 3**: Handlers, routers, FSM, middlewares
- **Mini Apps**: Web Apps inside Telegram
- **Payments**: Telegram Stars, subscriptions, paid media
- **Authentication**: Login Widget, URL Authorization
