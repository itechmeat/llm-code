# Skills Version Tracker

Quick reference for tracking tool versions across skills. Use this file to check for updates and plan skill maintenance.

## How to Use

1. Open this file when checking for skill updates
2. Visit the **Releases** link for each skill to check latest version
3. Compare with **Skill Version** column
4. If outdated, update the skill and this tracker

## Notes

- **—** indicates version/date not tracked for internal specs or skills without tracked upstream release metadata

## Update Procedure

When updating a skill:

1. Fetch release notes from **Releases** link
2. Compare current and latest versions
3. Read documentation for new features
4. Update skill files as needed
5. Update this tracker with new version/date
6. Add entry to `/CHANGELOG.md`
